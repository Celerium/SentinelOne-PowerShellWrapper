# Get-SentinelOneBaseURI

## SYNOPSIS
Shows the SentinelOne base URI global variable.

## SYNTAX

```powershell
Get-SentinelOneBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-SentinelOneBaseURI cmdlet shows the SentinelOne base URI global variable value.
