# Get-S1BaseURI

## SYNOPSIS
Shows the S1 base URI global variable.

## SYNTAX

```powershell

```

## DESCRIPTION
The Get-S1BaseURI cmdlet shows the S1 base URI global variable value.

### EXAMPLE 1
